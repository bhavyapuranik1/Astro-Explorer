# 🌌 Astro Explorer

Astro Explorer is a modern astronomy web application featuring:

- 🌠 Interactive Sky Map
- ☄️ Near-Earth Asteroid Explorer
- 🚀 NASA Astronomy Picture of the Day
- 🤖 Astro AI Assistant
- 🔐 Google Authentication
- ☁️ Cloud Chat History
- 📎 File & Image Attachments

# 💻 Local Development

## Install Vercel CLI

```bash
npm install -g vercel
```

## Login

```bash
vercel login
```

## Clone Project

```bash
git clone https://github.com/bhavyapuranik1/Astro-Explorer.git
```

## Open Project

```bash
cd Astro-Explorer
```

## Add Environment Variable

```bash
vercel env add OPENROUTER_API_KEY development
```

Paste your OpenRouter API Key when prompted.

## Run

```bash
vercel dev
```

Open:

```
http://localhost:3000
```
